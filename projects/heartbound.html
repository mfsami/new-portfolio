Heartbound.

Spring 2024

Stack: GODOT, GDSCRIPT

Heartbound is a tiny 2D platformer I created back in 2024. One of the first games I ever built. It’s insanely short but intentionally tight, a little love letter to the feeling I had right after finishing Celeste. I made it in the Godot engine with GDScript.

The core loop is classic platforming. Precise jumps, hazardous collisions, collectibles, and readable level beats with level progression. I kept the scope tiny so I could polish the basics. It was my first real pass at structuring a game project end-to-end which included learning input handling, player state, hazards, animations, and a couple of simple set pieces.

Since then I’ve fallen hard for Unity, but building Heartbound in Godot was a great experience. It taught me to pick a small target, finish it, and ship.
